# Campos de entrada por RF — Crear / Actualizar

Documento: tabla con los campos que el sistema espera en los endpoints de creación y actualización.

> Nota: la lista agrupa los RFs relacionados con operaciones `crear` y `actualizar` detectados en el backend.

|     RF | Acción                                 | Parámetros de ruta/query         | Body (campos de entrada)                                                                                                                                                                                                               |
| -----: | -------------------------------------- | -------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|   RF#1 | Autenticar usuario                     | —                                | `email`, `password`                                                                                                                                                                                                                    |
|   RF#3 | Registrar usuario                      | —                                | `email`, `password`                                                                                                                                                                                                                    |
|   RF#5 | Restablecer contraseña                 | —                                | Escenario token: `token`, `new_password`                                                                                                                                                                                               |
|        |                                        |                                  | Escenario autenticado: `current_password`, `new_password`                                                                                                                                                                              |
|   RF#8 | Actualizar perfil propio               | —                                | `email` (opcional)                                                                                                                                                                                                                     |
|   RF#9 | Asignar rol de usuario                 | `user_id` (path), `role` (query) | —                                                                                                                                                                                                                                      |
|  RF#14 | Crear dominio                          | —                                | `name`, `description` (opcional)                                                                                                                                                                                                       |
|  RF#15 | Actualizar dominio                     | `domain_id` (path)               | `name` (opcional), `description` (opcional)                                                                                                                                                                                            |
|  RF#21 | Crear Feature Model (FM)               | —                                | `name`, `description` (opcional), `domain_id`                                                                                                                                                                                          |
|  RF#23 | Actualizar FM                          | `model_id` (path)                | `name` (opcional), `description` (opcional), `is_active` (opcional)                                                                                                                                                                    |
|  RF#28 | Crear versión de modelo                | `model_id` (path)                | `source_version_id` (opcional), `description` (opcional)                                                                                                                                                                               |
|  RF#37 | Guardar UVL                            | `model_id`, `version_id` (path)  | `uvl_content`                                                                                                                                                                                                                          |
|  RF#39 | Aplicar UVL a estructura               | `model_id`, `version_id` (path)  | `uvl_content`                                                                                                                                                                                                                          |
|  RF#44 | Crear configuración persistente        | —                                | `name`, `description` (opcional), `feature_model_version_id`, `feature_ids` (lista)                                                                                                                                                    |
|  RF#47 | Actualizar configuración               | `id` (path)                      | `name` (opcional), `description` (opcional), `feature_ids` (opcional)                                                                                                                                                                  |
|  RF#54 | Crear característica                   | —                                | `name`, `type`, `feature_model_version_id`, `parent_id` (opcional), `group_id` (opcional), `resource_id` (opcional), `properties` (opcional)                                                                                           |
|  RF#62 | Actualizar característica              | `feature_id` (path)              | `name` (opcional), `type` (opcional), `parent_id` (opcional), `group_id` (opcional)                                                                                                                                                    |
|  RF#98 | Crear grupo de características         | —                                | `group_type`, `parent_feature_id`, `min_cardinality` (opc, default=1), `max_cardinality` (opcional)                                                                                                                                    |
|  RF#99 | Actualizar grupo de características    | `group_id` (path)                | `group_type` (opcional), `parent_feature_id` (opcional), `min_cardinality` (opcional), `max_cardinality` (opcional)                                                                                                                    |
| RF#104 | Crear relación de características      | —                                | `type`, `source_feature_id`, `target_feature_id`                                                                                                                                                                                       |
| RF#105 | Actualizar relación de características | `relation_id` (path)             | `type` (opcional), `source_feature_id` (opcional), `target_feature_id` (opcional)                                                                                                                                                      |
| RF#110 | Crear restricción                      | —                                | `feature_model_version_id`, `expr_text`, `description` (opcional)                                                                                                                                                                      |
| RF#111 | Actualizar restricción                 | `constraint_id` (path)           | `expr_text` (opcional), `description` (opcional)                                                                                                                                                                                       |
| RF#115 | Crear etiqueta                         | —                                | `name`, `description` (opcional)                                                                                                                                                                                                       |
| RF#118 | Crear recurso                          | —                                | `title`, `type`, `content_url_or_data`, opcionales: `description`, `language`, `duration_minutes`, `version`, `status`, `publication_date`, `author_name`, `owner_id`, `license`, `valid_until`, `tags`, `accessibility_notes`         |
| RF#119 | Actualizar recurso                     | `resource_id` (path)             | todos opcionales: `title`, `type`, `content_url_or_data`, `description`, `language`, `duration_minutes`, `version`, `status`, `publication_date`, `author_name`, `owner_id`\*, `license`, `valid_until`, `tags`, `accessibility_notes` |

\* Nota: `owner_id` en actualización de recurso solo puede modificarse por superusuarios.
